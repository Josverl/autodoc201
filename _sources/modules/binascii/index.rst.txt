   Binary/ASCII conversions.

   |see_cpython_module| :mod:`python:binascii`.

   This module implements conversions between binary data and various
   encodings of it in ASCII form (in both directions).

.. py:function:: a2b_base64(data) -> bytes

   Decode base64-encoded data, ignoring invalid characters in the input.
   Conforms to `RFC 2045 s.6.8 <https://tools.ietf.org/html/rfc2045#section-6.8>`_.
   Returns a bytes object.


.. py:function:: b2a_base64(data, *, newline=True) -> bytes

   Encode binary data in base64 format, as in `RFC 3548
   <https://tools.ietf.org/html/rfc3548.html>`_. Returns the encoded data
   followed by a newline character if newline is true, as a bytes object.


.. py:function:: hexlify(data, sep: Optional[Any] = None) -> bytes

   Convert the bytes in the *data* object to a hexadecimal representation.
   Returns a bytes object.

   If the additional argument *sep* is supplied it is used as a separator
   between hexadecimal values.


.. py:function:: unhexlify(data) -> bytes

   Convert hexadecimal data to binary representation. Returns bytes string.
   (i.e. inverse of hexlify)
